Return and Replaceable

Basic return and replacement information for orders on Dealify.

Eligibility

  • Return/replacement request should be raised within 7 days of delivery.
  • Item should be unused, with original tags and packaging.
  • Category-wise exceptions may apply for hygiene or custom products.

How to Request

  • Go to My Orders and select the delivered item.
  • Choose Return/Replace and select a reason.
  • Upload photos if the product is damaged or wrong.
